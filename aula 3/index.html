Objetos { }

Os objetos são estruturas de dados que nos permitem armazenar dados de diversos tipos, utilizando chaves {}

  • Os objetos não garantem uma ordem específica. Eles não são ordenados.
  • Podem conter múltiplos tipos de dados.
  • Utilizam chaves {} para a sua declaração
  • As propriedades são acessadas usando ponto .

Array [ ]

Array são estruturas semelhantes a listas, e podem conter diversos tipos de dados. Inclusive guardar objetos dentro de um array.

  • Os elementos em um array [] são ordenados e cada item possui um índice numérico. Começando do zero 0 [ 0 1 2 3 4...]
  • Podem conter múltiplos tipos de dados.
  • Os elementos são acessados usando o índice numérico
