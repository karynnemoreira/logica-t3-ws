Function

Uma função é um bloco de código que realiza uma tarefa específica, e pode ser reutiizada.

Funções podem receber dados externos através de parâmetros ()

Parâmetros e Argumentos

Os parâmetros () de uma função são espaços reservados para receber um valor.

Os argumentos são os valores passados para a função.

Função anônima

É uma função que não tem nome, ou seja, vocÊ cria uma função e não dá nome pra ela.

Arrow Function

É uma expressão de função de seta. É uma alternativa compacta a expressão de função tradicional.
