Objetos {}

Os objetos são estruturas de dados que nos permitem armazenar dados e funcionalidades, Não são ordenados.

Array []

Array são estruturas semelhantes a listas e podem conter diversos tipos de dados, inclusive podemos guardar objetos {} dentro de um array. Array são ordenados.

Array trabalha com índice numérico. [0,1,2,3,4...]

Métodos de array

Método é um gatilho de uma função que já tem uma funcionalidade embutida. Métodos de inclusão e exclusão de itens.
