Operadores Aritméticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Resto da divisão

Operadores de comparação

  • = atribuição
  • == Verifica se os valores são iguais
  • === Verifica se o valor e o tipo são iguais
  • != Verifica se os valores são diferentes
  • !== Verifica se o valor e o tipo são diferentes
  • > Maior que
  • < Menor que
  • >= Maior ou igual
  • <= Menor ou igual
  • Em valores binários, 0 é falso (false) e 1 é verdadeiro (true)

Operadores Lógicos

  • && (and - E) Seu resultado é verdadeiro (true) se ambos valores forem verdadeiros
  • || (or - OU) É verdadeiro (true) se um dos valores for verdadeiro.
  • ! (not) Inverte o valor que é dado a ele
